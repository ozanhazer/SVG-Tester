Notes About Path Commands

  • Coordinates are absolute if the command is upper-case and relative if lower. Case doesn't matter for the commands that are not taking parameters (eg Z)
  • Cubic Bezier: C x1,y1 x2,y2 x,y
    x1,y1 first control point,
    x2,y2 second control point,
    x,y end point
  • Move to: M x y
  • Close Path: Z
  • Arc: A rx ry x-axis-rotation large-arc-flag sweep-flag cx cy
    large-arc-flag: Can take 0 or 1 as a value. 1: Arc angle is more than 180, 0: Less than 180
    sweep-flag: Can also take 0 or 1 as a value. 1: clockwise, 0: anti-clockwise
    See this to understand how it works: http://www.w3.org/TR/SVG/images/paths/arcs02.png
